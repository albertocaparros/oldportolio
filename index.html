Tecnologias Web

A pesar de haber trabajado en distintos campos, mi foco principal desde que terminé mis estudios ha sido el desarrollo web.

Mi primer proyecto importante fue durante mis prácticas en el departamento de venta de software de IBM en Belgica, para el software de oficina Lotus Notes. Creamos un portal con la tecnología de uso privado de IBM: xPages, que utiliza estandares web (HTML + CSS) para la interfaz de usuario y el framework de Javascript Dojo Toolkit para la lógica interna.

Mas recientemente, he trabajado como analista y desarrollador de aplicaciones hibridas en el ayuntamiento de El Ejido (Almería,España). Usando el framework Ionic, desarrollé varias aplicaciones para el uso interno de la administración y para los ciudadanos del municipio. Todos los proyectos fueron creados para funcionar en distintas plataformas (Web, Android, iOS), y constaban de una aplicación creada con Ionic (Angular2) y sus respectivas APIs REST en PHP.
